# SOM DENTAL, Implant & Facial Skin Clinic — Demo PWA

Minimal, human-first landing page for SOM Dental — same design language and pattern as the Praxoma demo, adapted for a dental & facial skin clinic.

## What's included
- `index.html` — the whole page (Hindi/English toggle, WhatsApp + Call CTAs, concern chips, services, doctors, address)
- `manifest.json`, `sw.js`, `icons/` — installable, works offline once opened
- `assets/clinic-exterior.jpg` — your clinic's own building photo (from the uploaded flyer)

## Numbers wired in
- **WhatsApp** (concern chips + main button): +91 89570 39512
- **Call** button: +91 75204 68910
- **Address card** links to Google Maps for directions

## Host on GitHub Pages
1. Push all these files (keeping the `icons/` and `assets/` folders) to a new repo.
2. Settings → Pages → deploy from the branch/folder with these files.
3. Live at `https://<username>.github.io/<repo-name>/`.

## Note on content
Doctor names, degrees, services and address are taken directly from the clinic's own flyer you shared. Double-check the credentials text before going live — a few (e.g. "Associate Professor, DAV Dental College") were on the flyer but left out of the compact doctor cards; happy to add them back in if you want fuller bios.
